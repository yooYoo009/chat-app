# Chat App

このプロジェクトは、簡単なリアルタイムチャットアプリケーションです。

## 必要な環境
- Node.js (推奨バージョン: 16.x 以上)
- npm (Node.js に含まれています)

## セットアップ手順
1. プロジェクトのフォルダをダウンロードまたはクローンします。
2. ターミナルでプロジェクトのフォルダに移動します。
3. 以下のコマンドを実行して依存関係をインストールします：

npm install

4. サーバーを起動します：

npm start

5. ブラウザで以下の URL にアクセスします：

http://localhost:3000

## 注意事項
- サーバーを起動している間、ターミナルを閉じないでください。
- 他のユーザーとチャットするには、同じネットワーク内でこの URL を共有してください。
